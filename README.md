Add Whatsapp fixed button for your website.
It is very simple to use.

- You must use jQuery.
- Download fixedwp.js and include in your page. (Just below jquery)

Basic Usage;
```html
$('body').fixedwp({
  number: '+1234567890',
  text: 'Hi! How are you?'
});
```
All Options;
```html
$('body').fixedwp({
    number: "+1234567890",
    text: "Hi. How are you?",
    bg: "#25D366",
    color: "#fff",
    placement: "right",
    align_right:  "30px",
    align_bottom: "30px",
    align_top: "auto",
    align_left: "auto",
    padding: "17px"
});
```
That is all!
